# YandexMusicLinkExport
Экспортирует песни с вашего плейлиста в формате: АВТОР - НАЗВАНИЕ (ССЫЛКА)
1. Откройте место откуда нужно скопировать песни, так, чтобы были видны треки.
2. Нажмите F12 и откройте консоль
3. Если она блокирует вставку - allow pasting
4. Всталяете скрипт и запускаете.
5. Далее по инструкции UI.
